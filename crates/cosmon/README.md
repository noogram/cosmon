# cosmon

This crate **reserves the `cosmon` name** on crates.io. It contains no code.

[Cosmon](https://docs.noogram.org) is a stateless CLI that gives AI agents a
persistent identity, a typed lifecycle, and crash-recovery — so you can run ten
agent sessions in parallel without losing track of who is doing what. It is
distributed as a single binary named **`cs`**, not as a Rust library.

## Install the `cs` binary

Prebuilt binaries (Linux, macOS) are on every
[GitHub Release](https://github.com/noogram/cosmon/releases).

From source:

```sh
cargo install --git https://github.com/noogram/cosmon cosmon-cli
```

See <https://docs.noogram.org> for documentation.
