`POST /v1/molecules/{id}/collapse`

Usage: cosmon-remote molecule collapse [OPTIONS] --reason <REASON> <ID>

Arguments:
  <ID>  

Options:
      --profile <PROFILE>  Profile name (default: `default_profile` from config.toml, or `$COSMON_REMOTE_PROFILE`)
      --reason <REASON>    
      --cause <CAUSE>      
      --json               Render machine-readable JSON instead of human text where applicable
      --token <TOKEN>      Bearer JWT (overrides `$COSMON_REMOTE_TOKEN`). When unset and an authenticated call is made, the CLI mints a token via the profile's `oidc_url`
  -h, --help               Print help
