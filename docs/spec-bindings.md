## The triple bijection

Cosmon's specification lives in three mutually-consistent forms. A clause is correctly specified iff all three agree.

```
        prose (CLAUDE.md / CONSTITUTION.md)
                    │
                    ▼
   scenario TOML (tests/spec/scenarios/*.toml)
                    │
                    ▼
   Lean proposition (Track C, foundry_proposition)
```

Prose tells a human *what* the invariant is. The scenario TOML declares an *executable* witness — a fixture the harness can replay to detect regressions. The Lean proposition is the *formal* statement the polymer-proof correspondence (Track C) will discharge. `cs test --binding-report` is the fsck that checks every clause has all three.
