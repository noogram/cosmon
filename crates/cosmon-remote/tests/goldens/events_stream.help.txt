`GET /v1/events` — connect to the SSE stream and print each event to stdout. The stream runs until killed (Ctrl-C)

Usage: cosmon-remote events stream [OPTIONS]

Options:
      --molecule-id <MOLECULE_ID>      Filter to a single molecule id (server-side query filter). When omitted, every event for the tenant noyau is streamed
      --profile <PROFILE>              Profile name (default: `default_profile` from config.toml, or `$COSMON_REMOTE_PROFILE`)
      --json                           Render machine-readable JSON instead of human text where applicable
      --last-event-id <LAST_EVENT_ID>  Resume from a known last-seen event id (HTTP `Last-Event-ID` header)
      --token <TOKEN>                  Bearer JWT (overrides `$COSMON_REMOTE_TOKEN`). When unset and an authenticated call is made, the CLI mints a token via the profile's `oidc_url`
  -h, --help                           Print help
