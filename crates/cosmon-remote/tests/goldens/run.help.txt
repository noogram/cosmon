Like `do`, then price it: brackets the same nucleate + tackle + follow flow with two `GET /v1/quota` reads and reports the quota delta THIS run charged against your bucket. Zero new routes; the leak caveat is printed honestly

Usage: cosmon-remote run [OPTIONS] <TOPIC>

Arguments:
  <TOPIC>  What you want done — becomes the molecule's `topic` variable

Options:
      --formula <FORMULA>
          Formula to nucleate (the standard one-shot work unit) [default: task-work]
      --profile <PROFILE>
          Profile name (default: `default_profile` from config.toml, or `$COSMON_REMOTE_PROFILE`)
      --json
          Render machine-readable JSON instead of human text where applicable
      --kind <KIND>
          
      --token <TOKEN>
          Bearer JWT (overrides `$COSMON_REMOTE_TOKEN`). When unset and an authenticated call is made, the CLI mints a token via the profile's `oidc_url`
      --var <KEY=VALUE>
          Free-form variables `--var k=v`, repeat as needed
      --tag <TAGS>
          
  -y, --yes
          Skip the credit guard for this run (scripts/CI). Does NOT persist consent — a script's yes is not the operator's
      --follow-timeout <FOLLOW_TIMEOUT>
          Give-up deadline (seconds) for the follow phase. The worker keeps running server-side past it; `result <id>` later [default: 1800]
      --poll-interval <POLL_INTERVAL>
          Poll cadence (seconds) of the follow phase [default: 5]
      --no-events
          Disable the live events tail (the observe poll still runs)
  -h, --help
          Print help
