# crates/cosmon-rpp-adapter/data/surface_events.txt
#
# Append-only surface event log. Each line is one `surface_added` event:
# a molecule mounted an HTTP route on the RPP and pinned it into the
# §8p frozen surface (ADR-080).
#
# This file is the single source of truth for `frozen_api_surface()` —
# AND, since task-20260610-06a4 (delib-20260610-9a0c K2), for the
# per-route principal, minimal scope, §8p exposure classification, and
# one-line blurb. The parser lives in `crates/cosmon-surface-canon`;
# `cosmon-rpp-adapter/build.rs` and `cosmon-thin-cli/build.rs` fold it
# at compile time into const tables. Counter assertions derive from
# `cargo build`, not from hand-edited numbers.
#
# Discipline (vg-20260523-a682, Phase 1 / Commit 3 — torvalds Option D';
# wheeler I-ADDITIVE-COUNTERS invariant):
#   1. ADD: append a new line at the end. Never reorder.
#   2. NEVER edit an existing line — events are immutable. (Exceptions
#      on record: task-20260610-06a4 performed the one-pass controlled
#      column migration 4 → 7 fields; task-20260610-0b57 flipped the
#      exposure column of the `POST /v1/avatar/converse` line
#      adapter-only → tenant-verb (delib-20260610-9a0c T3 — the verb
#      joined the §8p bijection). In both cases the (METHOD PATH,
#      molecule_id, date) triples were byte-preserved.)
#   3. NEVER renumber, never bump a count — the test reads the file.
#   4. A merge of two branches each appending one event yields two
#      adjacent diff lines and merges cleanly.
#
# Format: METHOD PATH | molecule_id | YYYY-MM-DD | principal | scope | exposure | blurb
#
#   principal ∈ {tenant, operator, worker} — wire-form principal class
#     (mirrors the `#[verb]` annotation on the matching stub).
#   scope — minimal OAuth2 scope required by the route: `-` for
#     authentication-level routes (no scope check), else one or more
#     `cosmon:`-prefixed scopes joined by `+` (AND semantics).
#   exposure ∈ {tenant-verb, adapter-only, operator-only} — the §8p
#     classification. `tenant-verb` routes MUST have a matching
#     `#[verb]` annotation (bijection-tested); `adapter-only` routes
#     intentionally have none; `operator-only` must never appear on
#     the frozen surface (asserted by api_surface_freeze.rs).
#   blurb — one-line human description (drained from the clap
#     doc-comments). Required.
#
# Lines starting with `#` and blank lines are ignored.
# The seven `|`-separated fields are trimmed; `|` is not allowed inside
# any field. The fold REFUSES ambiguous lines (unknown principal or
# exposure, malformed scope, empty blurb) — the build fails.
#
