## Dr. Stone (manga/anime)

**The parallel**: In Dr. Stone, all of humanity is petrified for 3,700
years. Senku (a genius scientist) wakes people one by one and rebuilds
civilization from scratch, following a technology tree.

In cosmon: agents are "petrified" by token limits, crashes, or reboots.
`⚛ COSMON RESUME` wakes them up. They check the state of the world and
resume building.

| Dr. Stone | Cosmon |
|-----------|--------|
| Petrification | Token limit / crash |
| Revival fluid | `⚛ COSMON RESUME` signal |
| Senku's roadmap | `fleet.toml` specification |
| Kingdom of Science | The fleet |
| Assigning roles by talent | Agent roles (writer, reviewer, architect) |
| "Ten billion percent!" | `cargo test --workspace` all green |
| Stone Wars | Preemption (`cs preempt`) |

**Key difference**: Senku rebuilds from zero. Our agents have RESILIENCE
briefing — they check existing state and resume, not restart from scratch.
Also, our petrification lasts 30 seconds, not 3,700 years.

**Why it resonates**: Dr. Stone is fundamentally about using science to
rebuild civilization through organization and specialization. Cosmon is
about using typed state machines to organize AI agents through roles and
communication. Both are "physics of coordination."
