# sovereignty-allowlist.txt — frozen vocabulary of what MAY travel in
# dist/avatar-tenant-demo/ (buterin's allow-list, not a deny-list of tokens).
#
# Every lowercase alphabetic token (length >= 4) present in the tracked
# bundle at snapshot time. sovereignty-gate.sh extracts the live token set
# and WARNs on any token NOT listed here — so a NEW proper noun (a galaxy
# name, an operator first name, a coined identifier) that no shaped DENY
# class models still surfaces for human review the moment it appears.
#
# Regenerate deliberately after a legitimate vocabulary change:
#     ./scripts/sovereignty-gate.sh --update-allowlist
# The git diff of THIS file is the reviewable record of what was admitted.
# ---
